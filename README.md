# Dotfiles
This repository contains my personal configuration files.

## Configured apps
- neovim
- sway
- foot

## Dependencies
### sway
- wmenu
- gammastep
- pulseaudio
- brightnessctl
### neovim
- wl-clipboard
- clangd

## Setup guide
First, run the setup script: <br>
```./setup.sh``` <br>
<br>
Then, open neovim and run this command to install plugins: <br>
```:PlugInstall```
